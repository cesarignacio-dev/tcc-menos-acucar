# Projeto Menos Açúcar + Vida

**Descrição Técnica**
Sistema desenvolvido para análise nutricional e conscientização sobre o consumo de açúcar. A aplicação realiza cálculos de IMC e conversão de gramas de açúcar para medidas visuais, auxiliando na educação alimentar.

**Tecnologias e Infraestrutura**
* **Frontend:** HTML5, JavaScript e Tailwind CSS.
* **Backend:** Firebase (Auth & Firestore) para gestão de dados e autenticação.
* **Versionamento:** GitHub para controle de código-fonte e armazenamento de ativos.
* **Segurança:** Implementação de domínios autorizados para proteção de chaves de API.

**Organização de Arquivos**
O repositório está estruturado com diretórios independentes para imagens de produtos e arquivos de sistema, garantindo a escalabilidade e manutenção do catálogo.
