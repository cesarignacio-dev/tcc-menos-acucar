Projeto Menos Açúcar + Vida
​Descrição Estratégica
Sistema inteligente desenvolvido para monitoramento nutricional, com foco na conscientização do consumo de açúcar refinado através de cálculos de IMC e conversão visual de gramas em medidas caseiras.
​Arquitetura e Tecnologias
​Frontend: Interface desenvolvida em HTML5, JavaScript Vanilla e Tailwind CSS para design responsivo.
​Backend/Database: Integração com Firebase (Authentication e Cloud Firestore) para gestão de usuários e banco de dados em tempo real.
​Cloud Hosting: Aplicação hospedada via Replit (Produção) e GitHub Pages (Redundância).
​Versionamento de Ativos: Organização de mídias estruturada em diretórios específicos no GitHub para otimização de requisições.
